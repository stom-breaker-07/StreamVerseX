# StreamVerseX – Global OTT Streaming Intelligence Platform

## Overview

StreamVerseX is a production-style Data Engineering and Analytics project inspired by OTT platforms like Netflix, Amazon Prime Video, Disney+, and JioCinema.

The project analyzes streaming behavior, subscriptions, content performance, user engagement, search analytics, and CDN performance using PostgreSQL, Python, Power BI, and FastAPI.

---

## Tech Stack

- PostgreSQL
- SQL
- Python
- FastAPI
- Power BI
- Git & GitHub

---

## Datasets

- Video Streaming Sessions
- User Watch History
- Subscription Transactions
- Content Metadata
- CDN Buffering Logs
- App Clickstream Events
- Search Recommendation Logs
